# Tests Directory

This directory contains various test, debug, and utility scripts for the DailyCup application.

## Structure

- `debug/` - Debug scripts for troubleshooting issues
- `check/` - Scripts to check database structure, configurations, etc.
- `fix/` - Scripts to fix common issues
- `setup/` - Setup and initialization scripts
- `update/` - Scripts for updating data or configurations
- `verify/` - Verification scripts

## Security

This directory is protected by .htaccess and should not be accessible from the web.

## Usage

These scripts are for development and maintenance purposes only. Run them via command line with PHP.
